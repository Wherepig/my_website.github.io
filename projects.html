One Time Pad

Purpose:

Easy way to obscure short plaintext messages with a low algorithmic/time complexity

How this was achieved:

Create a new randomized cipher of ASCII characters for every message. Message could not be longer than the total number of ASCII characters used for cipher. Once the message was created the cipher would randomize the ASCII characters again for the next message.

Tools Used:

  • Python : For the base language
  • Regex : Pattern matching and decoding and encoding cipher text
  • ASCII Characters : basis for ciphertext

    • Storm alert

    Purpose:

    Predict storm/no-storm (binary) outcome using SGD Regression (Ai/ML)

    How this was achieved:

    A dataset of Sythetic storm alerts (yes/no) was loaded and parsed for the model. The data was split into two sets; True Labels and Predicted labels. An SGD Regressor model was applied to the training data ( predicted data ). After the model ran the data, a threshold was applied to the predicted output ( this took car of the binary nature of the storm alerts). Then, a Confusion matrix was applied to both the True Labels and Predicted Labels ( this would help see how far off or how accurate the model was in predicting the true data.)

    Tools Used:

  • Python : Prefered coding language for running complex Ai models
  • Google Colab : For runtime purposes and a centralized repository for python imports
  • Sythetic Weather Alerts : A spreadsheet of Data entries to test the Ai models
